What is Gauss-Kronrod Integration?

Many calculators provide a method of doing numerical integration at the touch of a button.  But what is going on in that black box?  To get an idea, let's review the standard numerical techniques first encountered in calculus.  

A simple method is the trapezoid rule, a linear approximation (degree 1).  The trapezoid rule will give an exact answer when integrating any linear segment of a function (degree 0 or 1).  Simpson's rule approximates the function with a parabola.  The quadratic approximation will give an exact answer when integrating any segment of a function that is degree 2, 1, or 0.  With either of these methods, you choose the values of x, then calculate the values of f(x), or you already know the values of f(x) if you are using experimentally acquired data.  Finding the integral does not depend on knowing the function; one can obtain an estimate of the integral directly from the data points.  However, finding the error of the approximated integral does depend on knowing the function.  Formulas for calculating the maximum error in the estimated integral for the trapezoid rule and Simpson's rule are determined theoretically using the second and fourth derivatives, respectively.  The error in the approximation is reduced by increasing the number of subintervals between the lower limit and upper limit of integration.  Both of these methods depend on knowing the x-values, which are usually at equally spaced intervals.

This contest was rigged; the example was selected to show the advantage of using the Gauss-Kronrod method.  Other examples could be selected so that the trapezoid rule or Simpson's rule could have a lower computational cost, depending on the function to be integrated and the error tolerance desired. The point is that Gauss-Kronrod integration is capable of producing numerical approximations quickly and accurately for many functions, and G7-K15 always has a computational cost of 15 (or multiples of 15 if the integration interval has to be subdivided to meet the error tolerance).  This makes it reliable, accurate, and fast for most of the integrals a calculator user would want to estimate.
